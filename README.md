Build using :

Maven 3.1.0 </br>
oraclejdk8 </br>
Spring MVC </br>

In order to execute all the tests, all you need to do is run:

    $ mvn clean package
    $ java -jar .\target\dependency\webapp-runner.jar .\target\*.war
